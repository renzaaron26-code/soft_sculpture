================================================================================
                      INSTALLATION & SETUP
================================================================================

STEP 1: Install Dependencies
  $ npm install

STEP 2: Set Up Database
  $ mysql -u root -p < DATABASE_SETUP.sql
  
  OR in MySQL Workbench:
  - File → Open SQL Script → DATABASE_SETUP.sql
  - Execute

STEP 3: Run Development Server
  $ npm run dev
  
  Visit: http://localhost:5173

STEP 4: Build for Production
  $ npm run build
  
  Output: dist/ folder ready for deployment

================================================================================
                    DEPLOYMENT OPTIONS
================================================================================

OPTION 1: Vercel (Recommended)
  1. npm run build
  2. vercel
  3. Configure deployment settings
  4. Push to production

OPTION 2: Netlify
  1. npm run build
  2. netlify deploy --prod --dir=dist

OPTION 3: Traditional Hosting
  1. npm run build
  2. Upload dist/ folder via FTP/SSH
  3. Configure web server for multi-page routing

OPTION 4: Docker
  1. Create Dockerfile
  2. Build image: docker build -t softsculpture .
  3. Run container: docker run -p 80:3000 softsculpture
