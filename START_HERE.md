## ✅ Quick Setup (TL;DR Version)

If you're in a hurry:

```bash
# 1. Extract and open project
unzip soft-sculpture-store.zip
cd soft-sculpture-store
code .

# 2. Install dependencies
npm install

# 3. Set up database (in VS Code MySQL extension)
# - Create database: soft_sculpture_db
# - Import: DATABASE_SETUP.sql

# 4. Run dev server
npm run dev

# 5. Open browser
# Go to: http://localhost:5173
```

Done! Site is running. 🎉
